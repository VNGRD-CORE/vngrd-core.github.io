# VNGRD// CORE

**Browser-native system for real-time audiovisual production**

![VNGRD// CORE Interface](./assets/vngrd-hero.gif)

---

## Overview

VNGRD// CORE is an experimental system exploring how real-time media production can be executed directly inside the browser.

The project was built to address a common problem:

> Creating audiovisual content typically requires multiple tools, heavy installations, and complex routing between audio, video, and streaming software.

VNGRD// investigates whether this workflow can be simplified into a single, browser-native environment.

---

## What You Can Do

- Capture live audio input  
- Generate audio-reactive visuals in real time  
- Record sessions directly in the browser  
- Export audiovisual content without external tools  

**Workflow:**
> Input → Visualize → Record → Export

All within a single interface — no installs required.

---

## Why This Matters

Modern media workflows are:

- fragmented  
- resource-heavy  
- dependent on multiple tools  

VNGRD// CORE explores a different approach:

> treating the browser as a real-time media execution environment

---

## Quick Start

**Launch the system:** https://vngrd-core.github.io

No installation required.  
Allow microphone/camera access and begin recording directly in the browser.
